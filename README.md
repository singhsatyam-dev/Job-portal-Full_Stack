# 🚀 CareerForge - AI Powered Job Portal

CareerForge is a full-stack MERN Job Portal that connects Job Seekers and Recruiters through an intuitive platform for job discovery, application management, and AI-powered resume analysis.

## 🌟 Live Demo

Frontend: https://job-portal-full-stack-jade.vercel.app/

Backend API: https://job-portal-career-forge.onrender.com/

## 📌 Features

### 👨‍💼 Job Seeker Features

- User Registration & Login
- Secure JWT Authentication
- Browse Available Jobs
- Search Jobs by Keywords
- View Detailed Job Information
- Apply to Jobs
- Track Applications
- AI Resume ATS Scoring

### 🏢 Recruiter Features

- Recruiter Registration & Login
- Create New Job Listings
- Edit Existing Jobs
- Delete Job Listings
- View Applicants
- Manage Recruitment Pipeline

### 🤖 AI Resume Scorer

CareerForge includes an AI-powered ATS (Applicant Tracking System) Resume Analyzer built using Google Gemini.

Users can:

- Upload Resume PDFs
- Enter a Target Job Role
- Receive ATS Match Score
- Get Personalized Improvement Suggestions
- Identify Missing Skills
- Improve Resume Relevance
